Kisi-Kisi Struktur Data

1. Definisi Struktur Data

Struktur data adalah cara mengorganisir, mengelola, dan menyimpan data agar bisa diakses dan dimodifikasi dengan efisien.

Contoh nyata: Struktur data seperti array dapat diimplementasikan dalam penyimpanan data siswa di sekolah berdasarkan urutan kelas.

2. Perbedaan Tipe Data, Variabel, dan Konstanta

  • Tipe Data: Menentukan jenis nilai, seperti int, float, char, atau boolean.
  • Variabel: Tempat untuk menyimpan data yang nilainya bisa berubah.
  • Konstanta: Tempat untuk menyimpan data yang nilainya tetap.

3. Array dalam Struktur Data

Array adalah struktur data yang menyimpan sejumlah elemen dengan tipe yang sama dalam satu blok memori. Elemen array diakses dengan indeks.

int angka[5] = {10, 20, 30, 40, 50};
int nilaiPertama = angka[0];  // Mengakses elemen pertama

4. Matriks dalam Struktur Data

Matriks adalah array dua dimensi yang menyimpan data dalam bentuk baris dan kolom. Perbedaan dengan array satu dimensi adalah array dua dimensi memiliki struktur tabel.

matriks = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];
nilai = matriks[1][2];  // Mengakses elemen pada baris 2 kolom 3

5. Algoritma Bubble Sort

Bubble Sort adalah algoritma pengurutan dengan cara membandingkan elemen berurutan dan menukarnya jika urutannya salah.

Kelebihan: Mudah dipahami.
Kekurangan: Kurang efisien untuk data besar (O(nÂ²)).

6. Linear Search dan Binary Search

  • Linear Search: Memeriksa elemen satu per satu.
  • Binary Search: Memecah data terurut menjadi dua bagian untuk pencarian lebih cepat (O(log n)).

7. Record dalam Struktur Data

Record adalah struktur data yang dapat menyimpan elemen data dengan tipe berbeda dalam satu entitas, seperti objek dalam OOP.

mahasiswa = {
    "nama": "Andi",
    "NIM": 123456,
    "tanggal_lahir": "2000-01-01"
}
